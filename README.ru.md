# Life Pilot Agent

**Система управления жизнью на базе AI, спроектированная практикующим психологом.**

Не очередной таск-менеджер. Life Pilot — это персональная операционная система, которая захватывает ваши мысли, управляет задачами и — самое главное — помогает рефлексировать, ставить осмысленные цели и реально их достигать. Коучинг-протокол построен на профессиональных психологических фреймворках, а не на шаблонных советах по продуктивности.

Построен на [Claude Code](https://docs.anthropic.com/en/docs/claude-code) + [MCP Protocol](https://modelcontextprotocol.io/) с интеграциями Google Calendar, Todoist и Obsidian.

> 🇬🇧 [English version](README.md)

## Зачем это нужно?

Большинство инструментов продуктивности решают не ту проблему. Они помогают организовать задачи — но задачи не являются узким местом. Узкое место — это понимание что действительно важно, способность заметить когда застрял, и вовремя скорректировать курс, пока месяцы не утекли впустую.

Life Pilot закрывает этот разрыв:

| Типичный инструмент | Life Pilot |
|---|---|
| Хранит заметки | Захватывает → классифицирует → создаёт задачи → привязывает к целям |
| Вы проверяете календарь | Утренний брифинг приходит автоматически |
| Обзор? Какой обзор? | Еженедельный дайджест с коучинговыми вопросами |
| Цели записал и забыл | Ежемесячные сессии планирования через диалог |
| Нет системы рефлексии | GROW-коучинг: неделя → месяц → квартал → год |
| Шаблонные промпты | Вопросы разработаны практикующим психологом |

**Один Telegram-бот. Голос в приоритете. Работает 24/7 на вашем VPS.**

## Идея проекта

Этот проект создан практикующим психологом, который увидел паттерн у клиентов: людям не хватает не амбиций и не интеллекта — им не хватает последовательной системы рефлексии и самокоррекции.

Три принципа, которые определили дизайн:

**1. Захват без трения.** Голосовое в Telegram → готово. Никаких приложений, никаких форм. AI сам классифицирует, создаёт задачи, сохраняет. Ваша задача — просто говорить.

**2. Ритм важнее мотивации.** Утренний план. Вечерний отчёт. Недельная рефлексия. Месячный обзор. Квартальная проверка. Годовой цикл. Система создаёт ритм — а ритм создаёт прогресс, даже когда мотивация угасает.

**3. Профессиональный коучинг, автоматизированный.** GROW-протокол — это не чат-бот трюк. Банк вопросов разработан психологом — поддерживающий тон, без оценки, распознавание паттернов, мягкая конфронтация с избеганием. AI адаптирует вопросы к вашим реальным данным: конкретные цели, просроченные задачи, повторяющиеся блокеры.

## Что умеет

### 📥 Захват всего подряд
Отправляйте что угодно в Telegram — агент сам решит что с этим делать:
- **Голосовые** → транскрипция (Groq Whisper, бесплатно) → классификация → сохранение или задача
- **Текст** → парсинг на задачи, идеи, находки
- **Фото** → сохранение с контекстом
- **Пересланные сообщения** → извлечение и категоризация

### 📋 Умное управление задачами
- Автосоздание задач в **Todoist** с правильными проектами и приоритетами
- Сортировка входящих мыслей по категориям: идеи, находки, проекты, рефлексии
- Обнаружение устаревших задач и напоминания
- Отслеживание переносов и паттернов просрочек

### 📅 Ритм дня
- **Утренний план** — автоматически приходит с событиями из календаря, задачами и приоритетами
- **Вечерний отчёт** — что было захвачено и обработано за день
- **Недельный дайджест** — обзор прогресса с ключевыми метриками
- **Месячное планирование** — интерактивная сессия постановки целей через вопросы

### 🧭 GROW-коучинг
Сердце Life Pilot. Структурированная система рефлексии на основе фреймворка GROW (Goal → Reality → Options → Will), спроектированная практикующим психологом.

**Как работает:**
1. AI анализирует ваши данные — цели, выполненные/просроченные задачи, паттерны, предыдущие рефлексии
2. Выбирает 2-4 вопроса из курируемого банка, адаптированных к вашей ситуации
3. Вопросы приходят по одному — отвечаете текстом или голосом (несколько сообщений на вопрос)
4. AI подводит итоги и предлагает конкретные обновления целей
5. Вы подтверждаете — цели обновляются автоматически

**Циклы коучинга:**
- **Еженедельный** (суббота) — что сработало, что нет, энергетические паттерны, фокус на следующую неделю
- **Ежемесячный** (1-е число) — актуальность целей, пересмотр приоритетов, анализ паттернов, оценка ресурсов
- **Квартальный** — глубокий анализ годовых целей, коррекция курса, отсечение того что не работает
- **Годовой** — декабрь: благодарность, уроки, итоги года. Январь: выравнивание с видением, новые цели, первые шаги

**Чем отличается от шаблонных опросов:**
- Вопросы психологически выверены — поддерживающий тон, без оценки, распознавание паттернов
- AI не задаёт случайные вопросы — выбирает на основе ваших реальных просрочек, пропущенных целей, повторяющихся блокеров
- Отложенные вопросы переносятся, пропущенные сессии отслеживаются — ничего не теряется
- 3 попытки напоминания на сессию — настойчиво, но не навязчиво
- Все рефлексии хранятся в структурированном markdown для долгосрочного самопознания

### Coach Mode — диалоговый коучинг
Интерактивные сессии через кнопку "Коуч" или `/coach`:
- Свободный диалог с Claude, заземлённый в ваших целях и контексте
- Голосовые сообщения поддерживаются на протяжении всего разговора
- Кнопка **Zoom In** — когда теряетесь в абстракциях, получаете конкретные следующие шаги
- Кнопка **Zoom Out** — когда погрязли в деталях, возвращает к большой картине
- "стоп" завершает сессию — система предлагает сохранить инсайты в vault
- История разговора хранится до 10 обменов в сессии

### Free Chat — прямой диалог с Claude
Кнопка "Чат" открывает прямую линию к Claude:
- Спрашивайте что угодно — поиск по заметкам, советы, мозговой штурм, анализ
- Полный доступ к контексту vault и истории коучинга
- Без жёсткого протокола — просто разговор с AI-ассистентом
- Голосовые сообщения поддерживаются

### Undo-система
Каждое действие AI (создание задачи, обновление цели, запись в vault) сопровождается кнопкой Undo:
- Окно отмены 5 минут для любого AI-изменения
- Кнопка автоматически исчезает по истечении времени — не засоряет чат
- Устаревшие Undo-пейлоады очищаются из памяти

| 2 | План — Неделя — Статус |

| 4 | Коуч — Чат — Помощь |

### Стек

- **Язык:** Python 3.12+
- **Менеджер пакетов:** uv (astral.sh)
- **Telegram-фреймворк:** aiogram 3.0+ (async)
- **Транскрипция:** Groq Whisper API (whisper-large-v3-turbo, бесплатный тир)
- **Задачи:** Todoist API
- **AI-движок:** Claude Code CLI (subprocess)
- **MCP-серверы:** Todoist, Google Calendar
- **Хранение:** файловая система (Obsidian vault, Markdown + JSONL-сессии)
- **Деплой:** systemd на Ubuntu VPS
- **Качество кода:** ruff + mypy strict + pytest

## Быстрый старт

### Что понадобится

| Компонент | Зачем | Стоимость |
|---|---|---|
| Claude Pro/Max | AI-агент | $20/мес |
| VPS (не РФ/РБ) | Бот работает 24/7 | ~$5/мес |
| GitHub | Бэкап и синхронизация | Бесплатно |
| Groq | Транскрипция голоса (Whisper) | Бесплатный тир |
| Todoist | Управление задачами | Бесплатно / $4/мес Pro |

### 1. Клонируйте и настройте

```bash
git clone https://github.com/YOUR_USERNAME/life-pilot-agent.git
cd life-pilot-agent
cp .env.example .env
```

Заполните `.env` вашими ключами:

```env
TELEGRAM_BOT_TOKEN=       # От @BotFather
GROQ_API_KEY=             # console.groq.com (бесплатно, транскрипция Whisper)
TODOIST_API_KEY=          # Todoist → Settings → Integrations
VAULT_PATH=./vault
ALLOWED_USER_IDS=[123]    # Ваш Telegram ID (от @userinfobot)
GIT_PUSH_ENABLED=true
CLAUDE_TIMEOUT=1200       # Таймаут subprocess в секундах
TRANSCRIPTION_LANGUAGE=ru # Язык Whisper (ru, en и т.д.)
GOOGLE_TOKEN_PATH=~/life-pilot/token.json  # OAuth-токен Google Calendar
```

### 2. Заполните цели

Файлы в `vault/goals/`:

- `0-vision-3y.md` — видение на 3 года
- `1-yearly-2026.md` — цели на год
- `2-monthly.md` — приоритеты на месяц
- `3-weekly.md` — фокус на неделю

### 3. Установите и запустите

```bash
# Установка зависимостей
uv sync

# Локальный запуск
uv run python -m life_pilot

# Или деплой через systemd (продакшен)
sudo cp deploy/*.service deploy/*.timer /etc/systemd/system/
sudo systemctl enable --now life-pilot-bot.service
sudo systemctl enable --now life-pilot-process.timer    # Вечерняя обработка в 21:00
sudo systemctl enable --now life-pilot-weekly.timer      # Недельный дайджест
```

### 4. Настройте расписание под себя

Все времена отчётов и коучинг-сессий настраиваются в `src/life_pilot/bot/main.py` → `create_scheduler()`. Настройте под свой ритм:

| Событие | Время по умолчанию | Что менять |
|---|---|---|
| Утренний план | 11:00 | cron hour в `send_morning_plan` |
| Вечерняя обработка | 21:00 | `life-pilot-process.timer` или job в scheduler |
| Месячный отчёт | 1-е число 20:30 | cron day/hour в `scheduled_monthly_report` |
| Недельный GROW-коучинг | Суббота 21:00 | cron day/hour в `scheduled_grow_weekly` (пропускает 1-3 числа) |
| Месячный GROW-коучинг | 1-3 числа 21:00 | cron day/hour в `scheduled_grow_monthly` |
| Квартальный GROW-коучинг | Варьируется | `scheduled_grow_quarterly` в grow_scheduler.py |
| Годовой GROW-коучинг | 20/23/26 дек, 5/10 янв | `scheduled_grow_yearly_*` в grow_scheduler.py |
| Проверка здоровья vault | Ср + Вс 22:00 | job `healthcheck` в scheduler |
| Таймзона | Europe/Kyiv | параметр `timezone` в `create_scheduler()` |

> ⚠️ **Важно:** Установите таймзону и время, которые соответствуют вашему распорядку. Система работает лучше всего, когда утренний план приходит до начала дня, а вечерний отчёт — после окончания рабочего дня.

### 5. Проверьте

```bash
# Логи бота
sudo journalctl -u life-pilot-bot -f

# Логи планировщиков
sudo journalctl -u life-pilot-process -f
sudo journalctl -u life-pilot-weekly -f

# Линтеры
uv run ruff check src/
uv run mypy src/
uv run pytest
```
